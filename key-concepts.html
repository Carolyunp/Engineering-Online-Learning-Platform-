How to approach a question:

  1. Mesh the Assembly
    • Place a node at both ends of the beam, any supports, any moments and decide whether to place nodes on point loads or to use work equivalent end loads.
  2. Vector of degrees of freedom x
    • Each node will have a DOF: \(\begin{bmatrix}x_1 \\ \theta_1 \end{bmatrix} \)
    • If the node is:
      • Fixed support: \(\begin{bmatrix}0 \\ 0 \end{bmatrix} \)
      • Pinned/roller support: \(\begin{bmatrix}0 \\ \theta_1 \end{bmatrix} \)
      • No support: \(\begin{bmatrix}x_1 \\ \theta_1 \end{bmatrix} \)
  3. Global stiffness matrix K
    • Assemble the global stifness matrix using the formula for each element stiffness matrix.
  4. Joint load matrix AJ and equivalent load matrix AE and combined load matrix AC = AJ + AE
  5. Solve Kx = AC
  6. Find unknown reaction forces

These steps can be utilised to answer any question in Topic 2!
